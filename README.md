# PM Portfolio Skill · 产品经理作品集生成器

> 上传简历 → AI多轮追问 → 生成可视化编辑的HTML作品集 → 导出PDF

---

## 这是什么

一个部署在AI平台上的Skill，帮助产品经理从简历快速生成一份**专业、美观、可直接编辑**的HTML作品集。

输出是**单HTML文件**，在浏览器里打开后可以直接点击编辑所有内容，不需要任何安装或服务器。

---

## 快速开始

### 1. 部署Skill

将 `pm-portfolio.skill` 部署到你使用的AI平台：

| 平台 | 部署方式 |
|------|---------|
| **Hermes** | 放入 skills 目录，重启服务即可 |
| **OpenClaw** | 导入 Skill 文件 |
| **Claude.ai** | 上传为自定义指令 |
| **其他Agent** | 将 skill 内容作为系统提示词使用 |

### 2. 使用流程

```
你：上传简历PDF（或粘贴简历文字）
AI：分析简历，追问补充关键信息（2-3轮）
AI：生成HTML作品集文件
你：在浏览器打开，编辑修改内容
你：保存 / 导出PDF
```

### 3. 编辑作品集

下载HTML文件，用Chrome打开：

- 点击右下角 **✏️ 编辑** 进入编辑模式
- 点击任意文字直接修改
- 上传原型图片（编辑模式下点击图片区域）
- 增删卡片、步骤、标签等结构
- **Ctrl+S** 或点击 **💾 保存文件** 保存修改
- 点击 **📄 导出PDF** 生成PDF

## 编辑系统功能

### 可编辑的内容

打开编辑模式后，**所有文字都可以点击修改**，包括：

- 封面：姓名、引用语、标签、数字、联系方式
- 项目：标题、描述、步骤、指标、成果
- AI模块：工具描述、思考内容、判断标准
- GitHub项目：名称、描述、技术标签、链接
- 市场洞察：信号、判断、验证结果
- 职业轨迹：时间、公司、职位、描述

### 增删结构

| 类型 | 操作 |
|------|------|
| 项目卡片 | 右上角「× 删除」/ 底部「+ 新增项目」 |
| 步骤条目 | 悬停「×」删除 / 「+ 添加步骤」，序号自动递增 |
| AI工具/思考卡 | 右上角「×」/ 「+ 新增」 |
| 开源项目 | 右上角「×」/ 「+ 新增开源项目」 |
| 市场洞察 | 右上角「×」/ 「+ 添加」 |
| 职业轨迹 | 右上角「×」/ 「+ 新增职业经历」 |
| 标签 | 右上角红色「×」/ 末尾「+ 标签」 |
| 原型图区块 | 右上角「× 删除原型区块」 |

### 原型图上传

编辑模式下，点击图片区域（蓝色虚线框）上传图片。图片以 base64 内嵌在 HTML 里，单文件携带，不依赖外部存储。

## 保存机制

### 本地 / Hermes / OpenClaw 环境

点击「💾 保存文件」→ 自动下载包含修改内容的新 HTML 文件 → 用新文件替换旧文件。

> **为什么不能直接覆盖？**
> 浏览器安全机制不允许 JavaScript 直接写入本地文件，这是所有浏览器的底层限制。下载新文件替换是业界标准做法（VSCode Web 也用同样方式）。

### Claude.ai 等在线平台

保存写入浏览器 localStorage，刷新页面后自动恢复。如需永久保存，使用「📋 复制代码」粘贴到文本编辑器另存为 `.html` 文件。

### 数据持久化（三层优先级）

```
1. pe-saved-data 内嵌标签（本地保存的文件自带，最可靠）
2. localStorage（在线平台刷新后恢复）
3. HTML 原始内容（默认值）
```

## PDF导出

### 本地 / Agent 环境（推荐）

自动使用 html2canvas + jsPDF 直接生成 PDF 文件下载，效果最好。

### 在线平台

使用浏览器打印功能：
1. 点击「📄 导出PDF」
2. 打印对话框 → 目标选「**另存为PDF**」
3. 勾选「**背景图形**」（保留颜色和渐变）
4. 保存

## 常见问题

**Q：生成的HTML在哪里打开最好？**
A：Chrome 或 Edge，功能最完整，PDF效果最好。

**Q：编辑后刷新内容会丢失吗？**
A：不会。保存后数据写入 localStorage，刷新自动恢复。在本地环境下载新文件后，数据内嵌在文件里，永久携带。

**Q：可以给其他人分享HTML文件吗？**
A：可以直接发送 HTML 文件，对方打开后看到的是完整的作品集（预览模式，无编辑按钮干扰）。

**Q：PDF中背景颜色不显示怎么办？**
A：打印时记得勾选「背景图形」选项。

**Q：支持手机浏览器吗？**
A：作品集支持响应式，手机可以正常浏览。编辑功能建议在 PC 上操作。
