# pm-portfolio

产品经理作品集生成器。上传简历，AI 自动生成专业 HTML 作品集，可在浏览器里直接编辑、导出 PDF。

## 快速开始

```
1. 在 Hermes / OpenClaw / Claude.ai 中安装此 skill
2. 发送简历文件 + "生成产品经理作品集"
3. Chrome 打开生成的 HTML → Cmd+P → 另存为PDF
```

## 文件结构

```
SKILL.md              # skill 定义与工作流程
assets/
  portfolio-template.html  # 作品集模板（含编辑系统和 PDF 导出）
references/
  fields.md           # 字段映射
  design-system.md    # 设计规范
  editor-system.md    # 编辑系统参考
  addel-system.md     # 增删系统参考
```
