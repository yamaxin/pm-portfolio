# 字段完整说明

生成 JSON 时，以下是所有需要填充的字段。
每个字段的 `data-field` 是键名，填入当前用户的真实内容作为值。

**重要：**
- 字段名（键名）不能修改
- 所有值是纯文字，不含 HTML 标签
- 没有信息的字段写空字符串 `""`
- 数字字段（如 stat1-n）只写数字本身，不含单位
